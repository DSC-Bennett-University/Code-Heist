# Pattern Problem

Write a program to print the following pattern for a given number of lines.

## Input

Line represents number of lines(n) of pattern.

## Output

Pattern with n lines

## Sample Case

### Given Input
```py
5
```


### Output
```py

        * 
      * * 
    * * * 
  * * * * 
* * * * * 
```
