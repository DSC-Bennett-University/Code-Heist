# Trignometry

Write a program to find the solution of the below equation:

```py
2 * tan(x) + 3 * cos(x^2) + 19 * sin(x)
```

## Input

The value of `x`.

## Output

The solution of the above equation. Round to two digits if necessary.

## Sample case

### Input

```py
3.14
```

### Output

```py
-2.69
```

## Testcase 1

### Input

```py
-6
```

### Output

```py
5.51
```

## Testcase 2

### Input

```py
100
```

### Output

```py
-13.65
```

## Testcase 3

### Input

```py
45
```

### Output

```py
18.68
```
